Mail Configuration Instructions

For this form to work properly in Contact Form 7, make sure to configure the Mail tab with these settings:

  • To: Your email address
  • From: [your-email]
  • Subject: [your-subject]
  • Additional Headers: Reply-To: [your-email]
  • Message Body: Include all form fields ([your-name], [your-tel], etc.)

Also set up the spam filtering to validate the "What is 2+2=?" field to only accept "4".
